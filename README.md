# 🛒 Product Inventory API

## 📌 Description
This is a Product Inventory Management API built using ASP.NET Core Web API.  
It allows users to manage products with full CRUD operations.

---

## 🚀 Features
- Add new product
- Get all products
- Get product by ID
- Update product
- Delete product

---

## 🛠️ Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger UI

---

## 📂 Project Structure
- Controllers
- Models
- Data (DbContext)
- Services

---

## ▶️ How to Run
1. Clone the repo
2. Open in Visual Studio
3. Run the project
4. Open Swagger UI
